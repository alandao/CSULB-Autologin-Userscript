CSULB Autologin Userscript
-------------------------
Edit csulbLogin.user.js before installing. Fill in your username and password variables.

Firefox installation:
	Install greasemonkey add-on. Load csulbLogin.user.js into firefox. 
Chrome installation:
	Enable developer mode: Menu -> tools -> extensions -> (developer mode)
	Drag csulbLogin.user.js into the extensions page.
